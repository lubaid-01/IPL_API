IPL Dataset API Overview :

Description:

The API provides access to a comprehensive dataset related to the Indian Premier League (IPL), allowing users to retrieve various data points in JSON format. The API is designed to facilitate easy access to IPL statistics, player information, match details, and other relevant data for developers, analysts, and cricket enthusiasts.

Endpoints :

  • /info:

    Returns information about a data e.g names of lpayers, stadiums etc.

  • /player/player_name:

    Retrieves information about players, including statistics, team affiliations, and profiles. e.g base_url/player/V Kohli

  • /playervenue/player/venue:

    Returns statistics of a player at a venue. e.g base_url/V Kohli/Chenna Swami Stadium

  • /team/team_name:

    Provides data on all IPL teams, including team members, history, and performance metrics. e.g base_url/team/Royal Challengers Bangalore

  • /venue:

    Returns information about different venues used in the IPL, including location and capacity. e.g base_url/Chenna Swami Stadium

  • /season/year:

    Returns information about a season. e.g base_url/2019

  • /pvp/batsman/bowler:

    Returns statistics of a batsman against a bowler e.g base_url/V kohli/J Bumrah

  • /vs/team1/team2:

    Returns information about all matches played between the teams season and date wise. e.g base_url/Delhi Daredevils/Rajasthan Royals

Data Format:

All responses from the API are in JSON format, making it easy to integrate with web applications and other services. Each endpoint returns structured data, ensuring users can easily parse and utilize the information.

Usage:

Developers can use the API to build applications, conduct data analysis, or enhance cricket-related platforms. The API supports various query parameters to filter and sort the data based on user requirements.
