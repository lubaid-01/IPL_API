Description:

The API provides access to a comprehensive dataset related to the Indian Premier League (IPL), allowing users to retrieve various data points in JSON format. The API is designed to facilitate easy access to IPL statistics, player information, match details, and other relevant data for developers, analysts, and cricket enthusiasts.

Endpoints:

Available Endpoints

Endpoint Description Example
/docs Interactive API documentation (Swagger UI) /docs
/team/{team_name} Team roster and performance metrics /team/Royal Challengers Bangalore
/player/{player_name} Player statistics, profile, and team affiliations /player/V Kohli
/playervenue?player=&venue= Performance for a player at a specific venue /playervenue?player=V Kohli&venue=m chinnaswamy stadium
/venue/{venue_name} Venue details (location, capacity) /venue/m chinnaswamy stadium
/season/{year} Season summary and stats /season/2019
/pvp?batsman=&bowler= Batsman vs bowler statistics /pvp?batsman=V Kohli&bowler=JJ Bumrah
/vs?team1=&team2= Head-to-head match history between two teams /vs?team1=Delhi Daredevils&team2=Rajasthan Royals

Data Format:

All responses from the API are in JSON format, making it easy to integrate with web applications and other services. Each endpoint returns structured data, ensuring users can parse it easily.

Usage:

Developers can build applications, perform data analysis, or create cricket-related tools. Query parameters allow filtering and sorting based on user requirements.
